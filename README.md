# QingHen AI

QingHen AI is a browser-local AI image humanizer and realism enhancer for creators, ecommerce sellers and content teams preparing AI-generated images for real-world publishing.

## Features

- Browser-local image preview, tuning and export
- AI-looking artifact smoothing controls
- File-signal review for common metadata markers
- Platform presets for X, Instagram, Douyin, RedNote, WeChat Official Account and custom exports
- SEO/GEO static pages, sitemap, robots and llms.txt
- Manual review contact panel with WeChat and email

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build script regenerates SEO pages before running Vite.
